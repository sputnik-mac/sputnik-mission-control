# 🛰️ Sputnik Mission Control

Personal AI dashboard for OpenClaw + Sputnik agent.

## Features
- Real-time chat with Sputnik via Gateway WebSocket
- Status monitoring: Gateway, Qdrant, Ollama
- Memory count from Qdrant
- Clean dark UI

## Run
```bash
npm install
node server.js
```

Open http://localhost:3100
